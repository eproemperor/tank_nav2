🎯 各包的核心作用
包名	作用	类比
robot_bt_decision_maker	高层任务决策（你的DecisionMakerNode）	🧠 大脑决策层
robot_bt_navigator	Nav2的BT导航器（执行导航行为树）	🦶 执行层
robot_behavior_tree	导航行为树的XML配置文件	📋 导航流程说明书
robot_msgs	自定义消息类型（地图信息等）	📦 通信数据结构
robot_serial	与底层硬件（MCU、传感器）通信	🔌 硬件接口层
robot_utils	通用工具函数（不依赖其他模块）	🛠️ 工具箱



机器人开机
    ↓
robot_bring_up (启动所有节点)
    ↓
┌─────────────────────────────────────────────────┐
│  robot_bt_decision_maker (你的DecisionMakerNode) │
│  作用：高层决策（去哪、做什么）                    │
│  - 加载 AnswerBT.xml                            │
│  - 决定导航目标点                                │
└─────────────────┬───────────────────────────────┘
                  │ 发送目标点（NavigateToPose Action）
                  ↓
┌─────────────────────────────────────────────────┐
│  robot_bt_navigator (BT Navigator)              │
│  作用：执行导航行为树                            │
│  - 加载 nav2_tree_nodes.xml                     │
│  - ComputePathToPose (调用规划器)               │
│  - FollowPath (调用控制器)                      │
└─────────────────┬───────────────────────────────┘
                  │ 速度指令
                  ↓
┌─────────────────────────────────────────────────┐
│  robot_serial (串口通信)                        │
│  作用：与单片机通信                              │
│  - 发送速度控制指令                              │
│  - 接收传感器数据                                │
└─────────────────┬───────────────────────────────┘
                  │ 串口数据
                  ↓
              机器人底盘
